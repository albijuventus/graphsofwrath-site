About GridLab

GridLab is an interactive research tool for exploring Hamiltonian cycles and paths on rectangular grid graphs. Built with Three.js, it provides 3D visualization and editing capabilities for graph theory research.

Developed for research into Hamiltonian path and cycle reconfiguration on rectangular grid graphs.

What You Can Do

  • Generate Hamiltonian paths and cycles on 2D/3D rectangular grids
  • Verify if a configuration is Hamiltonian
  • Reconfigure using switch and backbite moves
  • Color and decorate edges and vertices to mark a subgraph of the grid
  • Add directional arrows to orient edges
  • Add notes to each grid for documentation
  • Export and import your work as JSON files
